# GlowBeauty Website Project

## 📁 Project Structure
```
glowbeauty-project/
├── index.html          # Main website file
├── images/            # Folder for product images
│   ├── rose-serum.jpg
│   ├── coconut-cream.jpg
│   ├── green-tea-cleanser.jpg
│   └── hibiscus-mask.jpg
└── README.md          # This file
```

## 🎨 Features
- Beautiful pink/purple gradient color scheme
- Fully responsive design (mobile-friendly)
- Smooth animations and hover effects
- Product showcase with "Add to Cart" buttons
- Customer testimonials section
- Social media integration
- Professional footer with contact info

## 🖼️ Adding Images
1. Download your product images
2. Save them in the `images/` folder
3. Update the HTML file:
   - Find the emoji placeholders (🌸, 🥥, 🍃, 🌺)
   - Replace them with image tags:
   ```html
   <img src="images/your-image.jpg" alt="Product Name">
   ```

## 🚀 How to Use
1. Open `index.html` in any web browser
2. Customize product names, prices, and descriptions
3. Add your own images to the images folder
4. Update contact information in the footer

## 🎯 Recommended Image Sizes
- Product images: 800x800px (square format)
- File format: JPG or PNG
- Keep file sizes under 500KB for faster loading

## 💡 Tips
- Use high-quality product photos
- Maintain consistent image dimensions
- Optimize images before uploading
- Test on mobile devices
